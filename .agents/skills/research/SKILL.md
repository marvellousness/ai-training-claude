# Research

## Routing

Based on research type, load the appropriate reference:

| Research Type | Reference File |
|---------------|----------------|
| New feature/technology | `reference/research-new.md` |
| Bug fix/troubleshooting | `reference/research-bug-fix.md` |

## Usage

When user requests research, first determine if it's:
- **New thing** (feature, technology, best practices) → use `research-new.md`
- **Bug fix** (error, exception, crash, workaround) → use `research-bug-fix.md`

Then follow the methodology in the corresponding reference file.
